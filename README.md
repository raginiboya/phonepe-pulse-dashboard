# 📊 PhonePe Pulse Data Visualization Dashboard

## 📌 Project Overview
The PhonePe Pulse Data Visualization project analyzes digital payment trends across India using the PhonePe Pulse dataset. The project converts raw transaction data into meaningful insights through interactive dashboards.

This dashboard helps explore:
- Transaction patterns across states
- User engagement trends
- Insurance adoption
- Device usage patterns

The analysis helps identify growth opportunities and understand digital payment adoption across India.

---

## 🛠️ Technology Stack

- **Python**
- **Pandas**
- **MySQL**
- **Streamlit**
- **Plotly**
- **GitHub**

---

## 📂 Data Source

The dataset used in this project comes from the official PhonePe Pulse GitHub repository.

https://github.com/PhonePe/pulse

The dataset contains:
- Aggregated transaction data
- User statistics
- Insurance transaction data
- State and district level insights

---

## ⚙️ Project Workflow

1️⃣ Extract JSON data from the PhonePe Pulse repository  
2️⃣ Transform and clean the data using Python  
3️⃣ Store structured data in MySQL tables  
4️⃣ Perform analysis using SQL queries  
5️⃣ Build an interactive Streamlit dashboard  
6️⃣ Generate insights through visualizations  

---

## 📊 Dashboard Features

The Streamlit dashboard includes:

✔ Interactive India Map  
✔ State-level transaction analysis  
✔ User growth trends  
✔ Insurance adoption insights  
✔ Device usage analysis  
✔ Case study visualizations  

Users can filter data based on:
- Category (Transactions / Users / Insurance)
- Year
- Quarter

## 🔍 Key Insights

- Maharashtra and Karnataka show the highest transaction values.
- Xiaomi and Samsung dominate smartphone usage among PhonePe users.
- Digital payments are growing rapidly across India.
- Insurance adoption is increasing but still has strong growth potential.
- Urban regions show higher engagement compared to rural regions.
